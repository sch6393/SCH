gitmessage.txt
===

### 한국어
```
# 타입: 제목
# 제목은 50자 이내, "무엇"을 변경했는지 명확히, 문장 끝에 "." 금지
################################################
# feature  : 새로운 기능                       #
# fix      : 버그 수정                         #
# docs     : 문서 수정                         #
# style    : 코드에 영향을 주지 않는 사항 수정 #
# refactor : 코드 리팩토링                     #
# test     : 테스트 코드                       #
# chore    : 빌드, 패키지 설정 수정            #
################################################

# 본문 (구체적인 내용)을 작성
# 여러 줄의 메시지를 작성할 때는 "-"로 구분 (한 줄은 72자 이내)

# 꼬릿말 (footer)을 작성 (현재 커밋과 관련된 이슈 번호 추가 등)
# 예) Close #7

```

### 日本語
```
# タイプ: タイトル
# タイトルは50文字以内、「何」を変更したのかをはっきりする、文末に「.」は禁止
#####################################################
# feature  : 新しい機能                             #
# fix      : バグを修正                             #
# docs     : 文書を修正                             #
# style    : コードに影響を与えない事を修正         #
# refactor : コードリファクタリング                 #
# test     : テストコード                           #
# chore    : ビルド及ぶパッケージの設定を修正       #
#####################################################

# 本文（詳しい内容）を作成
複数の行のメッセージを作成する時は「-」で区切り（1行は72文字以内）

# フッター（footer）を作成（今のコミットと関わるイシュー番号の追加等）
# 例） Close #7

```
